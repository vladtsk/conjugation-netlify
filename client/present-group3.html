3rd Group Verbs in the Present Tense

The 3rd group is the most irregular in French. It includes verbs with endings like -ir, -ire, -tre, -dre, -oir, -ondre, -endre.

Common patterns and examples of verbs that follow these patterns

Verbs ending in -IR

Pattern 1: partir, sortir, servir, (se) sentir, dormir, mentir...

  • Je pars demain. (I'm leaving tomorrow.)
  • Je sors ce soir. (I'm going out tonight.)
  • Tu pars demain. (You're leaving tomorrow.)
  • Tu sors ce soir. (You're going out tonight.)
  • Il / elle / on part demain. (He / She / One is leaving tomorrow.)
  • Il / elle / on sort ce soir. (He / She / One is going out tonight.)
  • Nous partons demain. (We're leaving tomorrow.)
  • Nous sortons ce soir. (We're going out tonight.)
  • Vous partez demain. (You're leaving tomorrow.)
  • Vous sortez ce soir. (You're going out tonight.)
  • Ils/elles partent demain. (They're leaving tomorrow.)
  • Ils/elles sortent ce soir. (They're going out tonight.)
More examples
  • Je me sens bien. (I feel good.)
  • Les enfants ne dorment pas très bien. (The children don't sleep very well.)

Pattern 2: offrir, ouvrir, souffrir, couvrir, découvrir...

  • J'offre un cadeau. (I offer a gift.)
  • J'ouvre la porte. (I open the door.)
  • Tu offres un cadeau. (You offer a gift.)
  • Tu ouvres la porte. (You open the door.)
  • Il / elle / on offre un cadeau. (He / She / One offers a gift.)
  • Il / elle / on ouvre la porte. (He / She / One opens the door.)
  • Nous offrons un cadeau. (We offer a gift.)
  • Nous ouvrons la porte. (We open the door.)
  • Vous offrez un cadeau. (You offer a gift.)
  • Vous ouvrez la porte. (You open the door.)
  • Ils/elles offrent un cadeau. (They offer a gift.)
  • Ils/elles ouvrent la porte. (They open the door.)

Pattern 3: venir, revenir, devenir, tenir, maintenir...

  • Je viens d'Italie. (I come from Italy.)
  • Je le soutiens. (I support him.)
  • Tu viens d'Italie. (You come from Italy.)
  • Tu le soutiens. (You support him.)
  • Il / elle / on vient d'Italie. (He / She / One comes from Italy.)
  • Il / elle / on le soutient. (He / She / One supports him.)
  • Nous venons d'Italie. (We come from Italy.)
  • Nous le soutenons. (We support him.)
  • Vous venez d'Italie. (You come from Italy.)
  • Vous le soutenez. (You support him.)
  • Ils/elles viennent d'Italie. (They come from Italy.)
  • Ils/elles le soutiennent. (They support him.)

Pattern 4: courir, secourir, parcourir...

  • Je cours vite. (I run fast.)
  • Tu cours vite. (You run fast.)
  • Il / elle / on court vite. (He / She / One runs fast.)
  • Nous courons vite. (We run fast.)
  • Vous courez vite. (You run fast.)
  • Ils/elles courent vite. (They run fast.)

Verbs ending in -IRE

Pattern 1: lire, dire, traduire, conduire...

  • Je dis la vérité. (I'm telling the truth.)
  • Je conduis une voiture. (I drive a car.)
  • Tu dis la vérité. (You're telling the truth.)
  • Tu conduis une voiture. (You drive a car.)
  • Il / elle / on dit la vérité. (He / She / One is telling the truth.)
  • Il / elle / on conduit une voiture. (He / She / One drives a car.)
  • Nous disons la vérité. (We're telling the truth.)
  • Nous conduisons une voiture. (We drive a car.)
  • Vous dites la vérité. (You're telling the truth.)
  • Vous conduisez une voiture. (You drive a car.)
  • Ils/elles disent la vérité. (They're telling the truth.)
  • Ils/elles conduisent une voiture. (They drive a car.)

Pattern 2: écrire, décrire, (s')inscrire...

  • J'écris une lettre. (I'm writing a letter.)
  • Je m'inscris à un cours de français. (I'm signing up for a French course.)
  • Tu écris une lettre. (You're writing a letter.)
  • Tu t'inscris à un cours de français. (You're signing up for a French course.)
  • Il/elle/on écrit une lettre. (He's writing a letter.)
  • Il/elle/on s'inscrit à un cours de français. (He/she/one is signing up for a French course.)
  • Nous écrivons une lettre. (We're writing a letter.)
  • Nous nous inscrivons à un cours de français. (We're signing up for a French course.)
  • Vous écrivez une lettre. (You're writing a letter.)
  • Vous vous inscrivez à un cours de français. (You're signing up for a French course.)
  • Ils/elles écrivent une lettre. (They're writing a letter.)
  • Ils/elles s'inscrivent à un cours de français. (They're signing up for a French course.)

Verbs ending in -TRE

Pattern 1: mettre, promettre, remettre, (se) battre...

  • Je mets une veste. (I'm putting on a jacket.)
  • Tu mets une veste. (You're putting on a jacket.)
  • Il/elle/on met une veste. (He/she/one is putting on a jacket.)
  • Nous mettons une veste. (We're putting on a jacket.)
  • Vous mettez une veste. (You're putting on a jacket.)
  • Ils/elles mettent une veste. (They're putting on a jacket.)

Pattern 2: connaître, reconnaître, naître...

  • Je connais cette ville. (I know this city.)
  • Tu connais cette ville. (You know this city.)
  • Il/elle/on connaît cette ville. (He/she/one knows this city.)
  • Nous connaissons cette ville. (We know this city.)
  • Vous connaissez cette ville. (You know this city.)
  • Ils/elles connaissent cette ville. (They know this city.)

Verbs ending in -DRE

  • J'apprends le français. (I'm learning French.)
  • Je prends une photo. (I'm taking a photo.)
  • Tu apprends le français. (You're learning French.)
  • Tu prends une photo. (You're taking a photo.)
  • Il/elle/on apprend le français. (He/she/one is learning French.)
  • Il/elle/on prend une photo. (He/she/one is taking a photo.)
  • Nous apprenons le français. (We're learning French.)

  • Vous apprenez le français. (You're learning French.)
  • Vous prenez une photo. (You're taking a photo.)
  • Ils/elles apprennent le français. (They're learning French.)
  • Ils/elles prennent une photo. (They're taking a photo.)

Pattern 2: attendre, entendre, répondre, perdre, vendre…

  • J'attends le bus. (I'm waiting for the bus.)
  • Tu attends le bus. (You're waiting for the bus.)

Pattern 3: éteindre, peindre, joindre, rejoindre, se plaindre…

  • J'éteins la lumière. (I turn off the light.)
  • Je la rejoins plus tard. (I'll join her later.)
  • Tu éteins la lumière. (You turn off the light.)
  • Tu la rejoins plus tard. (You'll join her later.)
  • Il/elle/on éteint la lumière. (He/she/one turns off the light.)
  • Il/elle/on la rejoint plus tard. (He/she/one will join her later.)
  • Nous éteignons la lumière. (We turn off the light.)
  • Nous la rejoignons plus tard. (We'll join her later.)
  • Vous éteignez la lumière. (You turn off the light.)
  • Vous la rejoignez plus tard. (You'll join her later.)
  • Ils/elles éteignent la lumière. (They turn off the light.)
  • Ils/elles la rejoignent plus tard. (They'll join her later.)

Verbs ending in -OIR

Pattern 1: devoir, recevoir…

  • Je dois partir. (I have to go.)
  • Je reçois des messages. (I get messages.)
  • Tu dois partir. (You have to go.)
  • Tu reçois des messages. (You get messages.)
  • Il/elle/on doit partir. (He/she/one has to go.)
  • Il/elle/on reçoit des messages. (He/she/one gets messages.)
  • Nous devons partir. (We have to go.)
  • Nous recevons des messages. (We get messages.)
  • Vous devez partir. (You have to go.)
  • Vous recevez des messages. (You get messages.)
  • Ils/elles doivent partir. (They have to go.)
  • Ils/elles reçoivent des messages. (They get messages.)

Pattern 2: vouloir, pouvoir…

  • Je veux les aider. (I want to help them.)
  • Je peux laisser un message. (I can leave a message.)
  • Tu veux les aider. (You want to help them.)
  • Tu peux laisser un message. (You can leave a message.)
  • Il/elle/on veut les aider. (He/she/one wants to help them.)
  • Il/elle/on peut laisser un message. (He/she/one can leave a message.)
  • Nous voulons les aider. (We want to help them.)
  • Nous pouvons laisser un message. (We can leave a message.)
  • Vous voulez les aider. (You want to help them.)
  • Vous pouvez laisser un message. (You can leave a message.)
  • Ils/elles veulent les aider. (They want to help them.)
  • Ils/elles peuvent laisser un message. (They can leave a message.)

Here are two important verbs that don't exactly follow any of the patterns above: voir (to see), savoir (to know, to know how to).

  • Je vois une étoile. (I see a star.)
  • Je sais cuisiner. (I know how to cook.)
  • Tu vois une étoile. (You see a star.)
  • Tu sais cuisiner. (You know how to cook.)
  • Il/elle/on voit une étoile. (He/she/one sees a star.)
  • Il/elle/on sait cuisiner. (He/she/one knows how to cook.)
  • Nous voyons une étoile. (We see a star.)
  • Nous savons cuisiner. (We know how to cook.)
  • Vous voyez une étoile. (You see a star.)
  • Vous savez cuisiner. (You know how to cook.)
  • Ils/elles voient une étoile. (They see a star.)
  • Ils/elles savent cuisiner. (They know how to cook.)

Verbs ending in -OIRE

The verbs boire and croire follow 2 slightly different patterns:

  • Je bois de l'eau. (I drink water.)
  • Je crois que c'est vrai. (I think it's true.)
  • Tu bois de l'eau. (You drink water.)
  • Tu crois que c'est vrai. (You think it's true.)
  • Il/elle/on boit de l'eau. (He/she/one drinks water.)
  • Il/elle/on croit que c'est vrai. (He/she/one thinks it's true.)
  • Nous buvons de l'eau. (We drink water.)
  • Nous croyons que c'est vrai. (We think it's true.)
  • Vous buvez de l'eau. (You drink water.)
  • Vous croyez que c'est vrai. (You think it's true.)
  • Ils/elles boivent de l'eau. (They drink water.)
  • Ils/elles croient que c'est vrai. (They think it's true.)
